# **App Name**: LevelUp: Placement Slayer

## Core Features:

- Daily Quest Generation: Generates daily main and side quests based on the user's progress and weak areas, focusing on coding, DSA, aptitude, core subjects, and resume building.
- Performance Analytics Tool: Uses performance data to analyze weak areas and suggests recovery missions and focus areas, providing a placement readiness score using generative AI.
- Gamified Profile System: A profile system with name, title, level, rank, and stats (Discipline, Intelligence, Strength, Consistency, Focus) that increase based on task completion. Includes XP calculation and level-up mechanics.
- Streak and Punishment System: Tracks daily streaks with a reward system (Shadow Mode for 7+ days streak) and a punishment system (XP loss for broken streak, rank drop for inactivity).
- Ranking System: Tiered ranking system (E to Monarch) with rank promotion animations based on XP and consistency.
- Fire Quote System: Displays random motivational quotes after quest completion and generates one custom quote daily using generative AI, based on the user's performance.
- Progress Dashboard: Dashboard displaying XP progress, weekly consistency graph, heatmap calendar, stat distribution radar chart, and quest lists.

## Style Guidelines:

- Primary color: Neon Blue (#4D77FF) to capture the energy of progress and the neon feel of the source material.
- Background color: Dark Gray (#121212) to provide a dark dungeon-like setting, improving focus by reducing distractions.
- Accent color: Glowing Purple (#A663CC) to highlight key elements such as stats and level, enhancing visual appeal.
- Body and headline font: 'Space Grotesk' sans-serif for a modern, techy look that suits both headlines and body text. Code font: 'Source Code Pro' for displaying code snippets.
- Use glowing, sharp icons to represent stats, quests, and achievements.
- Dark-themed, dungeon-style layout with glowing stat panels and an animated XP bar at the top.
- Smooth level-up animations, rank promotion animations, and a fire aura effect when streak exceeds 7 days.
